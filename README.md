## Installation
### Install necessary libraries
```> sudo apt-get install libglfw3```

```> sudo apt-get install libglfw3-dev```

```> sudo apt-get install freeglut3```

```> sudo apt-get install freeglut3-dev```

### Build and Execute
```> sh install.sh  (OU  ./install.sh)```

```> ./OpenGL```
